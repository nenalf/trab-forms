Questões

  1. Na tag <input> com atributo type="text", para que serve o atributo "placeholder"?

    Uma amostra de texto que dá uma breve descrição do que deve ser escrito durante o preenchimento daquele espaço de formulário.

  2. Na tag <input> com atributo type="radio" ou type="radio", para que serve o atributo "checked"?

    Especifica que um campo deve ser pré selecionado ao carregar a página.

  3. Na tag <input> para que serve o atributo "value"?

    Especifica o valor de um campo, isto é, o que estará descrito nele.

  4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

    Ao colocar a tag <input type="radio"> especifíca-se que apenas uma opção poderá ser selecionada. Para que seja possível selecionar várias, usa-se o type="checkbox".

  5. Para que serve a tag <label>? Para que serve o atributo "for" nessa tag?

    Estabelecer qual será o "rótulo", isto é, nome referente para identificação da tag input, indicando ao usuário o que ele deve preencher naquele espaço input. Além disso, é útil para fins de acessibilidade, visto que leitores de tela lerão em voz alta o rótulo quando o usuário focar no elemento "input"; e ele alterna o botão de opção/caixa de seleção, indicando para o usuário que, no label, é um texto, não uma caixa de preenchimento.

    O "for" define para qual tag será aquele rótulo descrito.

  6. Na tag <option> para que serve o atributo "selected"?

    Estabele que uma opção estará pré-selecionada quando uma página for carregada.

  7. Nas tags de formulário para que serve o atributo "required"?

    Informa que o usuário deve preencher o campo com um valor antes de enviar o formulário, ou seja, coloca o input com esse atributo como sendo de preenchimento obrigatório.

  8. Na tag <form> para que serve o atributo "action"?

    Define a ação a ser executada quando o formulário for enviado, por exemplo, uma saída em php. Se não houver definição de action, o que foi escrito será gravado na mesma página onde foi preenchido.

  9. Na tag <form> para que serve o atributo "method"?

    especifica o método HTTP a ser usado ao enviar os dados do formulário. Os dados do formulário podem ser enviados como variáveis de URL (com method="get") ou como transação de postagem HTTP (com method="post").

  10. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag <form>?

    Atributo "post".

  11. Qual valor deve ser informado no atributo "method" na tag <form> em um formulário que será usado para fazer upload de arquivos?

    Metódo GET (que gera variáveis de url) ou POST (como transação de postagem HTTP). Também é possível obter como opção do VSCode o valor "dialog", que representa uma caixa de diálogo ou outro componente interativo, tal como um inspetor ou janela, mas esse valor não será explorado durante a matéria.
